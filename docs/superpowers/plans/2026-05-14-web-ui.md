# Web UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the Textual threads TUI with a browser-based founder console built as a localhost React+Tailwind SPA bundled into the FastAPI daemon, architected so future CLI domains can be added without rework.

**Architecture:** Single FastAPI process serves both `/api/v1/*` (existing) and `/` (new SPA static mount). The SPA fetches a bearer token from a new localhost-gated bootstrap endpoint and attaches it to every HTTP+SSE call. Three frontend layers: `lib/api/` (1:1 mirror of daemon routes) → `features/<domain>/` (one folder per CLI domain) → generic `components/`. Threads is the only feature surfaced in this milestone; `lib/api/` is pre-built for every existing route so the conventions generalize before they calcify.

**Tech Stack:** FastAPI + StaticFiles (existing) · Vite 5 · React 18 · TypeScript (strict) · Tailwind CSS 3 · React Router v6 · TanStack Query v5 · `@microsoft/fetch-event-source` · `react-markdown` + `remark-gfm` · React Hook Form · Vitest + React Testing Library + MSW v2

**Spec:** `docs/superpowers/specs/2026-05-14-web-ui-design.md`

**Repo state assumed:** Worktree at `.claude/worktrees/thread-ui` on branch `worktree-thread-ui`. Daemon code lives in `src/daemon/`, tests in `tests/`. No existing `web/` directory.

**TDD discipline:** Every task writes the failing test first. Every commit is one logical unit. Use `pytest tests/path -v` for Python and `npm test -- --run path` for Vitest.

**Conventions:**
- All `git commit` steps use HEREDOC bodies with the Co-Authored-By trailer.
- Python test runs use `uv run pytest`.
- Node commands run from `web/`.
- Every step that produces code must include the literal code; no "similar to Task N" shorthand.

Create `src/daemon/routes/auth.py`:

```python
"""Localhost-only bootstrap endpoint that hands the SPA the bearer token.

The web UI runs entirely on the founder's local machine. Browsers cannot read
``~/.opc/daemon.token`` themselves, so the SPA fetches it once on load via this
endpoint and stashes it in ``sessionStorage``. The endpoint refuses any peer
that isn't loopback so the token never escapes the host.

No reverse-proxy assumption: the daemon is the terminal hop, so
``request.client.host`` is the real peer. ``X-Forwarded-For`` is ignored.
"""
from __future__ import annotations

from fastapi import APIRouter, HTTPException, Request, status

from src.daemon import paths

router = APIRouter()

_LOCAL_HOSTS = {"127.0.0.1", "::1", "localhost"}


@router.get("/auth/bootstrap")
def bootstrap(request: Request) -> dict:
    peer = request.client.host if request.client else None
    if peer not in _LOCAL_HOSTS:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail={"code": "not_localhost", "peer": peer},
        )
    token = paths.read_token()
    if token is None:
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="daemon token file missing",
        )
    return {"token": token}
```

Create `src/daemon/routes/web_static.py`:

```python
"""Static SPA mount + SPA fallback.

In production the daemon serves the React build from ``web/dist/``. The mount
location is resolvable in three ways, in order:

1. ``OPC_WEB_DIST`` environment variable (used by tests).
2. ``<repo>/web/dist`` relative to this source file.
3. None — daemon renders a small placeholder telling the founder to run
   ``scripts/build_web.sh``.

The fallback handler exists so React Router can own all non-``/api/*`` paths.
Without it, a hard refresh on ``/orgs/foo/threads`` would 404.
"""
from __future__ import annotations

import os
from pathlib import Path

from fastapi import FastAPI, HTTPException, Request
from fastapi.responses import FileResponse, HTMLResponse
from fastapi.staticfiles import StaticFiles


_PLACEHOLDER_HTML = """<!doctype html>
<html><head><meta charset="utf-8"><title>OPC — web UI not built</title>
<style>body{font-family:system-ui;background:#0b0d10;color:#e6e6e6;padding:3rem;max-width:42rem;margin:0 auto;line-height:1.5}
code{background:#1a1d22;padding:.15rem .4rem;border-radius:.25rem}</style></head>
<body><h1>OPC web UI is not built</h1>
<p>Run the following from the repo root, then refresh:</p>
<pre><code>scripts/build_web.sh</code></pre>
<p>Or, for development, run <code>npm run dev</code> in the <code>web/</code> directory.</p>
</body></html>
"""


def _resolve_dist_dir() -> Path | None:
    override = os.environ.get("OPC_WEB_DIST")
    if override:
        p = Path(override)
        return p if p.is_dir() else None
    # ``src/daemon/routes/web_static.py`` → parents[3] is repo root.
    repo_root = Path(__file__).resolve().parents[3]
    candidate = repo_root / "web" / "dist"
    return candidate if candidate.is_dir() else None


def register(app: FastAPI) -> None:
    """Attach SPA static mount + fallback handler to the FastAPI app."""
    dist = _resolve_dist_dir()

    if dist is not None:
        # /assets/* (and other top-level static files) come from dist.
        app.mount("/assets", StaticFiles(directory=str(dist / "assets")), name="assets")

        index_path = dist / "index.html"

        @app.get("/", include_in_schema=False)
        def _spa_root() -> FileResponse:
            return FileResponse(str(index_path))

        @app.get("/{full_path:path}", include_in_schema=False)
        def _spa_fallback(full_path: str, request: Request) -> FileResponse:
            # ``/api/*`` is already matched by the API routers — anything that
            # reaches this handler is a SPA route, so always serve index.html.
            # FastAPI's router dispatches API routes first because they were
            # registered before this catch-all.
            if full_path.startswith("api/"):
                raise HTTPException(status_code=404)
            return FileResponse(str(index_path))
    else:

        @app.get("/", include_in_schema=False)
        def _placeholder_root() -> HTMLResponse:
            return HTMLResponse(_PLACEHOLDER_HTML)

        @app.get("/{full_path:path}", include_in_schema=False)
        def _placeholder_fallback(full_path: str) -> HTMLResponse:
            if full_path.startswith("api/"):
                raise HTTPException(status_code=404)
            return HTMLResponse(_PLACEHOLDER_HTML)
```
